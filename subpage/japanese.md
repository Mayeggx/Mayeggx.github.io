# 学日语

> 学日语一直都是梦想一般的东西，个人对于它的世俗价值期望不大但也许什么时候会有妙用，更希望能做一些字幕组翻译组之类无私的工作。

## 计划

> N1还是 N2，反正就是要考的，板上钉钉的事！

2. 安心学课本单词

   |  さ  |  し  |  す  |  せ  |  そ  |
   |  た  |  ち  |  つ  |  て  |  と  |
   |  な  |  に  |  ぬ  |  ね  |  の  |
   |  は  |  ひ  |  ふ  |  へ  |  ほ  |
   |  ま  |  み  |  む  |  め  |  も  |
   |  や  |  い  |  ゆ  |  え  |  よ  |
   |  ら  |  り  |  る  |  れ  |  ろ  |
   |      |      |      |      |      |

>  し-xi  す-si  ち-qi  つ-ci  ふ-fu

2. 浊音

   |  あ  |  い  |  う  |  え  |  お  |
   | :--: | :--: | :--: | :--: | :--: |
   |  が  |  ぎ  |  ぐ  |  げ  |  ご  |
   |  ざ  |  じ  |  ず  |  ぜ  |  ぞ  |
   |  だ  |  ぢ  |  づ  |  で  |  ど  |
   |  ば  |  び  |  ぶ  |  べ  |  ぼ  |
   
>  じ-ji  ず-zi  ぢ-ji  づ-zi 

3. 半浊音
   
   |  あ  |  い  |  う  |  え  |  お  |
   | :--: | :--: | :--: | :--: | :--: |
   |  ぱ  |  ぴ  |  ぷ  |  ぺ  |  ぽ  |
   
4. 长音

   1. あ段＋あ
   2. い段＋い
   3. う段＋う
   4. え段＋え、い
   5. お段＋お、う
   6. 片仮名＋ー

5. 促音　っ 停顿一拍

6. 拗音  き--り＋や、ゆ、よ

7. 声调

   1. 0型：不下降，从低到高：我哥
   2. 1型：第一个字后下降，从高到低：哥哥
   3. 2型：第二个字后下降：我哥哥
   4. 3型：第三个字后下降：我的哥哥
> 不同声调的单词具有不同的意思，应该加以重视

8. 片假名：用于外来词、动物、拟声词
