# 乐理
> 也没想学得很深，就多了解下，增强分析音乐的能力为先！

1. 歌曲构成：前奏-主体（A+B）-间奏（过门）-尾奏

2. 构成单元：乐汇-乐节-乐句-乐段、乐步-乐章-乐曲

3. 乐音：有规律和固定音高（琴、定音鼓）

   噪音：有规律和固定音高（鼓、锣）

4. 音符：

   1. 全音符-二分音符-四分音符-八分音符-十六分音符
   2. 全休止符-半休止符-四分休止符-八分休止符-十六分休止符
     ![常见音符](https://upload-images.jianshu.io/upload_images/189683-8458852bc139a392.png?imageMogr2/auto-orient/strip|imageView2/2/format/webp)
   3. 符头+符干+符尾
     ![音符组成](https://upload-images.jianshu.io/upload_images/189683-9cee7fd6b4c18ed6.png?imageMogr2/auto-orient/strip|imageView2/2/format/webp)

5. 唱名：do-re-mi-fa-sol-la-si

6. 音名：

   |  C   |      |  D   |      |  E   |  F   |      |  g   |      |  A   |      |  B   |
   | :--: | :--: | :--: | :--: | :--: | :--: | :--: | :--: | :--: | :--: | :--: | :--: |
   |  do  |      |  re  |      |  mi  |  fa  |      | sol  |      |  la  |      |  si  |
   |      |  #C  |      |  #D  |      |      |  #F  |      |  #G  |      |  #A  |      |
   |      |  bD  |      |  bE  |      |      |  bG  |      |  bA  |      |  bB  |      |
   |      | sol  |      |  la  |      |      |  do  |      |  re  |      |  mi  |      |

   

7. 浮点：延长1.5倍的时间

8. 音符的名称：

   |   G    |   A    |  B   |  C   |   D    |  E   |  F   |
   | :----: | :----: | :--: | :--: | :----: | :--: | :--: |
   | 下属音 | 下中音 | 导音 | 主音 | 上主音 | 中音 | 属音 |
